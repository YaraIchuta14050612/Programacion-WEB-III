6. Layout Modular

  • Divide la página en bloques independientes.
  • Ventaja: Flexible y fácil de reorganizar.
  • Desventaja: Puede ser complejo sin una buena jerarquía visual.

                <!DOCTYPE html>
                <html lang="es">
                
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Layout Modular</title>
                    <style>
                        .container {
                            font-family: 'Poppins', sans-serif;
                            display: flex;
                            flex-wrap: wrap;
                            gap: 10px;
                            padding: 10px;
                        }
                
                        .module {
                            background: lightblue;
                            padding: 20px;
                            flex: 1 1 calc(33% - 20px);
                            text-align: center;
                        }
                
                        @media (max-width: 600px) {
                            .module {
                                flex: 1 1 100%;
                            }
                        }
                    </style>
                </head>
                
                <body>
                    <div class="container">
                        <div class="module">Módulo 1</div>
                        <div class="module">Módulo 2</div>
                        <div class="module">Módulo 3</div>
                    </div>
                </body>
                
                </html>
