7. Layout Asimétrico

  • Rompe la simetría tradicional para un diseño más dinámico.
  • Ventaja: Más atractivo y moderno.
  • Desventaja: Puede dificultar la lectura si no se equilibra bien.

                <!DOCTYPE html>
                <html lang="es">
                
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Layout Asimétrico</title>
                    <style>
                        .container {
                            font-family: 'Poppins', sans-serif;
                            display: flex;
                        }
                
                        .box1 {
                            width: 70%;
                            background: lightpink;
                            padding: 20px;
                        }
                
                        .box2 {
                            width: 30%;
                            background: lightblue;
                            padding: 20px;
                        }
                    </style>
                </head>
                
                <body>
                    <div class="container">
                        <div class="box1">Sección grande</div>
                        <div class="box2">Sección pequeña</div>
                    </div>
                </body>
                
                </html>
