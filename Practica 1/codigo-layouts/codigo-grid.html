Layout Grid

5. Layout en Rejilla (Grid Layout)

  • Organiza los elementos en una estructura de cuadrícula.
  • Ventaja: Diseño ordenado y escalable.
  • Desventaja: Puede parecer rígido si no se usa con creatividad.

                <!DOCTYPE html>
                <html lang="es">
                
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Layout Grid</title>
                    <style>
                        .container {
                            font-family: 'Poppins', sans-serif;
                            display: grid;
                            grid-template-columns: repeat(3, 1fr);
                            gap: 10px;
                            padding: 10px;
                        }
                
                        .box {
                            background: lightcoral;
                            padding: 20px;
                            text-align: center;
                        }
                    </style>
                </head>
                
                <body>
                    <div class="container">
