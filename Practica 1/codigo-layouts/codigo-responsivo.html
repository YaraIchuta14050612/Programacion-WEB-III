3. Layout Responsivo

  • Se ajusta dinámicamente mediante media queries.
  • Ventaja: Mejor experiencia en múltiples dispositivos.
  • Desventaja: Requiere más trabajo de diseño y pruebas.

                <!DOCTYPE html>
                <html lang="es">
                
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Layout Responsivo</title>
                    <style>
                        .container {
                            font-family: 'Poppins', sans-serif;
                            width: 90%;
                            margin: auto;
                            background: lightgreen;
                            padding: 20px;
                        }
                
                        @media (max-width: 600px) {
                            .container {
                                width: 100%;
                                background: orange;
                            }
                        }
                    </style>
                </head>
                
                <body>
                    <div class="container">
                        <h2>Este es un layout responsivo</h2>
                        <p>Cambia de color y tamaño cuando la pantalla es pequeña.</p>
                    </div>
                </body>
                
                </html>
