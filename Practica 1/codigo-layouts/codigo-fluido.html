Layout Fluido

2. Layout Fluido o Líquido

  • Usa porcentajes para ajustar el tamaño de los elementos.
  • Ventaja: Se adapta a distintos dispositivos.
  • Desventaja: Puede generar espacios desproporcionados.

                <!DOCTYPE html>
                <html lang="es">
                
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Layout Fluido</title>
                    <style>
                        .container {
                            font-family: 'Poppins', sans-serif;
                            width: 90%;
                            margin: auto;
                            background: lightblue;
                            padding: 20px;
                        }
                    </style>
                </head>
                
                <body>
                    <div class="container">
                        <h2>Este es un layout fluido</h2>
                        <p>El ancho se ajusta al 90% del tamaño de la pantalla.</p>
                    </div>
                </body>
                
                </html>
