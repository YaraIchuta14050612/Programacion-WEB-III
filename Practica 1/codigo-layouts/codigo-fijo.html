Layout Fijo

1. Layout Fijo

  • Usa un ancho definido en píxeles.
  • Ventaja: Control preciso del diseño.
  • Desventaja: No se adapta a diferentes tamaños de pantalla.

                <!DOCTYPE html>
                <html lang="es">
                
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Layout Fijo</title>
                    <style>
                        .container {
                            font-family: 'Poppins', sans-serif;
                            width: 1000px;
                            margin: auto;
                            background: lightgray;
                            padding: 20px;
                        }
                    </style>
                </head>
                
                <body>
                    <div class="container">
                        <h2>Este es un layout fijo</h2>
                        <p>Su ancho es de 1000px y no cambia con el tamaño de la pantalla.</p>
                    </div>
                </body>
                
                </html>
